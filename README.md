# Campus Lost \& Found System

A web-based system developed to digitize campus lost and found reporting.

## Features

* User authentication
* Report lost items
* Report found items
* View items
* Update status
* Delete reports
* Image upload
* Secure validation

## Technologies Used

* Node.js
* Express
* MySQL
* HTML/CSS/JS

## Setup Instructions

1. Install dependencies
   - npm install
2. Configure environment variables
   - Create .env file using .env.example
3. Run server
   - node server.js
4. Visit this site

&nbsp;  - http://localhost:3000/index.html
