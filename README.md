# Campus Lost \& Found System

A web-based system developed to digitize campus lost and found reporting.

## Features

* User authentication
* Report lost items
* Report found items
* View items
* Update status
* Delete reports
* Image upload
* Secure validation

## Technologies Used

* Node.js
* Express
* MySQL
* HTML/CSS/JS

## Setup Instructions



1\. Install Node.js

2\. Install MySQL Server 8.0

3\. Create a database named web\_fa

4\. Import the provided database.sql file

5\. Configure .env file with database credentials

6\. Run npm install in terminal

7\. Run npm run dev in terminal

8\. Open http://localhost:3000
